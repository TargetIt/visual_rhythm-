Why Coalescing Matters 为什么合并访存重要

一个 Warp (32 threads) 中每个线程读取 4 字节 (float)，共需 32 × 4 = 128 bytes。

GPU 内存控制器以 128 字节 cache line 为单位传输。如果 32 个地址落在同一 cache line 内（合并访存），只需 1 次事务；

如果地址散布在 N 个 cache line 中（非合并），则需 N 次事务，浪费带宽。

Utilization = bytes_requested / bytes_transferred = 128 / (N_transactions × 128)

在 Naive SGEMM 中，访问 B 矩阵列（行优先存储的连续地址）是合并的；访问 A 矩阵行中跨列元素（stride = N）则不合并。
